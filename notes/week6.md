## Notes
### Python Basics
- C has prototypes, Python has **Modules** (libraries) + **Packages** (collection of modules)
- Import specific features of library using "," to list each feature (not "#include" whole library like in C)
- Get input via the input function (only strings)
- Override automatic newline using _end=""_ (named parameter)
- Use single OR double quotes for strings (be consistent)
- elif NOT elseif in Python
- == NOT strcmp() in Python
### Concatenation
- concatenation --> print(x + y) *does NOT add
- add using int() function (w/ user output as argument)
